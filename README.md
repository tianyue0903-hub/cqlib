# Cqlib

Cqlib 2.0 项目基于 Rust 开发核心功能，并为 Python 3.10+、C++11+、Java 11+ 提供支持。


| 文件夹          | 说明              | 版本           |
|--------------|-----------------|--------------|
| cqlib-core   | 核心 Rust 实现      | Rust 1.70+   |
| cqlib-python | Python（PyO3) 接口 | Python 3.10+ |
| cqlib-cpp    | C++（cxx) 接口     | C++11        |
| cqlib-java   | Java（jni-rs) 接口 | Java 8       |
